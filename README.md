Stratio bdt
===========

Stratio Acceptance Test library

 * Cucumber for test definition 
 * TestNG for execution
 * AsyncHTTP Client
 * Selenium 
 * JSONPath
 * AssertJ

Testing runtime to rule over Stratio's acceptance tests
